# UC01: Criar gráficos de estatística

## Descrição Geral
Este caso de uso descreve o processo de criação de gráficos pelo usuário. O sistema permite a criação de gráficos de linhas, barras e setores, possibilitando a inserção dos dados necessários para cada tipo de gráfico.

### Requisitos
RF01: Criar gráfico de linhas
-RF02: Criar gráfico de barras
-RF03: Criar gráfico de setores

### Atores
- Usuário

### Prioridade
 - Alta

### Pré-condições

O programa deve estar em execução.
O usuário deve selecionar um dos tipos de gráfico disponíveis.

### Frequência de uso

Alta

### Condição de entrada

O usuário solicita a criação de um novo gráfico.

## Fluxo Principal

1. O usuário seleciona o tipo de gráfico desejado (linhas, barras ou setores) no menu principal.
2. O sistema apresenta os campos necessários para o tipo de gráfico escolhido.
3. O usuário insere os dados do gráfico.
4. O usuário informa os títulos, descrições ou nomes dos eixos quando aplicável.
5. O usuário pode adicionar novos pontos, barras ou setores conforme necessário.

   5.1 Caso algum valor informado seja inválido ou esteja incompleto, o sistema exibe uma mensagem de erro,
   o usuário deve corrigir os dados informados.
6. O usuário solicita a geração do gráfico.
7. O sistema gera e exibe o gráfico solicitado.
8. O usuário pode retornar ao menu principal.

### Fluxo Alternativo

Não há.

### Pós condições

Não há.

## Regras de Negócio

1. No gráfico de linhas, os pontos devem possuir coordenadas X e Y válidas.
2. No gráfico de barras, cada barra deve possuir um nome e um valor associado.
3. No gráfico de setores, cada setor deve possuir um nome, uma cor e um valor.
4. Deve existir pelo menos um elemento (ponto, barra ou setor) para que o gráfico possa ser criado.
5. O sistema deve impedir a criação de gráficos com dados inválidos ou incompletos.
